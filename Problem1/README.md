  <h4>Description</h4>
  <p>
      If we list all the natural numbers below 10
      that are multiples of 3 or 5, we get 3, 5, 
      6 and 9. The sum of these multiples is 23.

      Find the sum of all the multiples of 3 or 5
      below 1000.
  </p>

---

* Pull from the upstream repository 
into your master branch to keep it in sync with 
the main project:

```bash
git pull upstream master
```

* Create a new branch with the command:

```bash
git switch -c fix-issue
```

* Navigate to the Problem1 directory from the root 
of your project. 

* Create a directory for your solution in the 
format:
```
Solution[problem number]-[your GitHub nickname]/
```

When you are sure your solution runs properly  
([check the answer](https://projecteuler.net)):
- Stage your changes with the command:

```bash
git add files-that-you-changed
```

- Commit your changes with the command:

```bash
git commit -m "Fixed issue"
```

- Push changes to your fork (your GitHub 
repository) with the command:

```bash
git push origin fix-issue
```

Check out [README.md](../README.md) for more 
detailed instructions on how to make a pull request.

For any queries, feel free to open an issue,
write to [Discussions](https://github.com/AlexandrAnatoliev/project-euler/discussions/23)
